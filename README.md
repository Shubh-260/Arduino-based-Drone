# Arduino-based-Drone
In this project, I made a quadcopter using Arduino
Components used- Arduino UNO, 350mm frame, 6inch propeller, 2250mah battery, 30A ESC, 1600kv motor, Flysky FS-CT6B transmitter and receiver, mpu-6050
